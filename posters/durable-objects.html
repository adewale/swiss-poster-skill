01 Anatomy / what's in oneLook inside

What lives inside one object.

any client  →  idFromName("standup") ↓
/room/standup the named instance
storage
SQLite, persistent, co-located. Lives where the code does. Transactional. Outlives hibernation.
memory
JavaScript class fields. Lasts until hibernation; re-hydrate from storage on wake.
inbox
Requests handled one at a time, in arrival order. No locks. No races. The single-threaded story.
alarms
Scheduled work. Survives sleep. ctx.storage.setAlarm().
sockets
WebSockets, hibernate-able. Idle connections cost nothing; wake on the next message.
▽   hibernates when idle  ·  wakes on the next request   ▽
