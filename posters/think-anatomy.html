Inside Project Think.

Cloudflare's harness on its Agents SDK. The agent doesn't run on infrastructure — it IS the infrastructure: each one is a Durable Object with its own SQLite, hibernation, and live client sync. Think is the opinionated loop on top.
An agent is a Durable Object extends Agent
// src/index.ts import { Agent, routeAgentRequest, callable } from 'agents'; export class Triage extends Agent<Env, State> { @callable() async assign(user: string) { this.setState({ ...this.state, owner: user }); // SQLite + live sync this.sql`INSERT INTO log VALUES (${user})`; await this.schedule('0 9 * * *', 'digest'); // cron, durable } } export default { fetch: (req, env) => routeAgentRequest(req, env) };
// wrangler.jsonc "durable_objects": { bindings: [{ name: "Triage", class_name: "Triage" }] }, "migrations": [{ tag: "v1", new_sqlite_classes: ["Triage"] }]
Skip the boilerplate loop: extend Think instead for an opinionated harness — SessionManager, agentic loop, stream resumption, workspace tools, ExtensionManifest.
