03 Build one / define an agent
An agent is a small, declarative object: pick a model, hand it tools and skills, give it a sandbox, describe the goal.
modelthe LLM that reasons
toolstyped actions it can call — your APIs
skillsexpertise it loads — SKILL.md
sandboxwhere it runs safely — local()
instructionsthe task, in plain words
— agents/triage.ts@flue/runtime
import { createAgent } from '@flue/runtime';
import { local } from '@flue/runtime/node';
import triage from '../skills/triage/SKILL.md'
  with { type: 'skill' };
import * as gh from '../tools/github.ts';

export default createAgent(() => ({
  model:   'anthropic/claude-sonnet-4-6',
  tools:   [...gh],
  skills:  [triage],
  sandbox: local(),
  instructions,  // triage a bug, end to end
}));
