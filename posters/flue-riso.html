Agent = Model + Harness
step 01 — the model
Reason
The model reads the state and decides the next move — not a fixed branch, a judgment made each turn.
step 02 — the harness
Act
The harness lets it call a tool and make a real, controlled change to the world around it.
step 03 — the loop
Observe
The result feeds back in. The loop repeats until the task is finished or it asks for help.
The model supplies the reasoning; the harness supplies the loop. You hand it a task, not a script — and let it run the cycle on its own.
