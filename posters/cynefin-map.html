Disorder
the central state
You don't yet know which domain you're in. Break it apart; assign the pieces.
Cynefin is a sense-making framework, not a categorization grid: the data shapes the model, not the other way around. First name the kind of problem — ordered or unordered — then adopt the response that domain rewards. The same situation can move between domains as it evolves.
