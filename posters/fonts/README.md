# URW Base35 fonts

Free clones of the PostScript 35, used as typographic specimens in
`output/poster.html`. The OTF files are **not** vendored — they are
fetched at render time from Artifex's `urw-base35-fonts` repo.

## Fetch them

```sh
./fetch.sh
```

That pulls 11 OTFs (~900 KB total) into this directory.
