The Stack

An agent is one column of layers — from intent at the surface down to the runtime at the base. Each stratum depends on the one beneath it.

Instructions
The goal in plain words — a desired outcome, not a script to follow.
Model
The LLM that reasons and picks the next action. anthropic/claude-sonnet-4-6.
Reasoning Level
How hard it thinks, set per profile for the task at hand.

Subagent

External tools over the Model Context Protocol.

Context / Memory

Local

Runs that survive failures and restarts and pick up where they left off.
Observability

Powered by Pi

Triggers

Read the column top to bottom and you read an agent whole: a model given instructions, equipped, holding state, walled by a sandbox, kept durable — all of it standing on the runtime.
