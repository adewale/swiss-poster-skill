which domain am I in?
Exploit · Complex → Complicated
A pattern holds; constraints harden; hand it to the experts.
Disrupt · Complicated → Complex
Constraints break; the expert answer stops working; probe again.
Split · Disorder → out
From the center, break the problem up and send each part to its domain.
Dive · ordered → Chaotic, on purpose
Push a stuck system into chaos to force something new, then catch it.
Impose · Chaotic → Clear
Act to stop the bleeding, set a rule, and hold it.
Collapse · Clear → Chaotic — the cliff
Run the old rule while the world changes and the floor drops.
the reading
Best practice fits the domain you're in now, not the one you were in last week. Re-sense often; the map is a snapshot of a moving thing.
